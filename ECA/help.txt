Usage: ecc.exe [options] file...
Options:
    --help      Show this message.

    -O0         No optimization.
    -O1         Compiler optimization (Default).
    -O2         Extra assembly optimization.

    -E          Output preprocessed code.
    -S          Output assembly.

    -win64      Build for Win64.
    -wos32      Build for WOS32.

    -Werror     Treat warnings as errors.
    -w          Ignore all warnings.

    -o <file>   Send output to <file>.
